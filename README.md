# AI CV Assistant

This project is an AI-powered CV and Job Application Assistant.

## Features

- **User Authentication:** Secure registration and login.
- **CV Management:** Upload and manage your CVs (PDF, DOCX, TXT).
- **Job Application Tailoring:** Input job descriptions and generate tailored application letters using AI.
- **Profile Management:** Update your user profile.
- **Modern Dashboard:** Intuitive interface for managing your career documents.

## Tech Stack

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io) (with MongoDB)
- [Tailwind CSS](https://tailwindcss.com) (shadcn/ui)
- [tRPC](https://trpc.io)

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up your `.env` file (see `.env.example`).
4.  Start the database (MongoDB Replica Set required).
5.  Run the development server: `npm run dev`

## Deployment

Follow the deployment guides for [Vercel](https://vercel.com/docs) or [Docker](https://docs.docker.com/) for more information.
